



# <a href="#4568" class="header-anchor">#</a> Fee

## <a href="#8632" class="header-anchor">#</a> Quote

LV2 HK market quotes and A-share LV1 market quotes are free for Chinese
mainland customers.  
For some variaties, you need to buy quotation cards before obtaining
market data. For more details of quotation cards prices, please click
[Quote Right](/moomoo-api-doc/en/intro/authority.html#5331) and go to
data store.
